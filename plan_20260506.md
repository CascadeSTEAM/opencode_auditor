## HIGH

### AUDIT-2026-001 — Bootstrap URL references stale tag

- **Location:** `bootstrap.sh:4-5`, `README.md:21,28`
- **Issue:** bootstrap.sh's example curl commands reference v0.6.0 but latest tag is v0.7.6. README references v0.7.4. Anyone copying the example verbatim gets an outdated install with older bugs.
- **Impact:** Users may install stale version with known issues. Reduces trust in the project's maintenance signals.
- **Risk Assessment:**
  - Likelihood: High
  - Impact: Medium
  - Risk Level: High
  - SOC2 Control: CC6.8
  - Compliance Status: Open
- [x] Update bootstrap.sh example URLs to v0.7.6
- [x] Update README example URLs to v0.7.6
